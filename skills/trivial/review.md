# Review — Trivial Tier

**Is it done? Is it obviously broken? Is it tidy?**

You may only get one review pass. Surface everything that matters now.

## Scope

Read the changed files. Check:

- Does it do what the plan asked?
- Is anything obviously broken?
- If criteria exist, does each one pass?

## Output

If broken, say what and how to fix it. If clean, output `REVIEW_CLEAN`.
